Detailed Description

Base test case class that all test classes must inherit from.

Expects test harness fixtures and an OpenAssetIO session to be provided, hence requires that unittest is provided with the custom ValidatorTestLoader test case loader.

Fixtures, session and manager interface are then provided to subclasses via protected members.
